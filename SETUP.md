# uFit — Complete Setup Guide

## Step 1: Install Dependencies

```bash
cd ufit
flutter pub get
```

---

## Step 2: Firebase Setup (Required for Auth)

### 2a. Create Firebase Project
1. Go to https://console.firebase.google.com
2. Click "Add project" → name it "ufit" → Continue
3. Enable Google Analytics (optional) → Create project

### 2b. Add Android App
1. In Firebase Console → Project Overview → Add app → Android
2. Package name: `com.example.ufit` (or your custom ID)
3. Register app → Download `google-services.json`
4. **Place `google-services.json` in `android/app/google-services.json`** (replace the placeholder)

### 2c. Add iOS App
1. In Firebase Console → Add app → iOS
2. Bundle ID: same as your Xcode bundle ID (e.g. `com.example.ufit`)
3. Download `GoogleService-Info.plist`
4. **In Xcode**: drag `GoogleService-Info.plist` into `Runner/` folder
5. Open `ios/Runner/Info.plist` and replace:
   - `YOUR_REVERSED_CLIENT_ID` → value from `REVERSED_CLIENT_ID` field in GoogleService-Info.plist
   - `YOUR_IOS_CLIENT_ID` → value from `CLIENT_ID` field

### 2d. Enable Auth Providers in Firebase Console
1. Firebase Console → Authentication → Sign-in method
2. Enable **Email/Password**
3. Enable **Google** → set support email → Save

### 2e. Auto-generate firebase_options.dart
```bash
# Install FlutterFire CLI (once)
dart pub global activate flutterfire_cli

# In your project root:
flutterfire configure
```
This generates `lib/firebase_options.dart` automatically.

### 2f. Uncomment Firebase init in main.dart
Open `lib/main.dart` and uncomment:
```dart
import 'firebase_options.dart';
// ...
await Firebase.initializeApp(options: DefaultFirebaseOptions.currentPlatform);
```

## Step 3: Google Sign-In Setup

### Android
1. In Firebase Console → Authentication → Sign-in method → Google → enable
2. In Firebase Console → Project Settings → Your apps → Android → add your **SHA-1** fingerprint:
   ```bash
   # Get debug SHA-1:
   cd android && ./gradlew signingReport
   ```
3. Download updated `google-services.json` and replace in `android/app/`

### iOS
- Already handled by `GoogleService-Info.plist` from Step 2c
- Make sure `CFBundleURLTypes` in `Info.plist` has your `REVERSED_CLIENT_ID`

---

## Step 4: Razorpay Setup (Android Payments)

1. Sign up free at https://razorpay.com
2. Dashboard → Settings → API Keys → Generate Test Key
3. Open `lib/services/payment_service.dart`
4. Replace `'rzp_test_YOUR_KEY_HERE'` with your actual key
5. For production: use `rzp_live_...` key and implement server-side verification

**Cost:** 2% per transaction (India) — no monthly fee

---

## Step 5: Google Play / Apple IAP (Optional — for store distribution)

### Google Play Billing
1. Create app in Google Play Console
2. Monetize → Products → Subscriptions → Create:
   - `ufit_pro_monthly` — ₹299/month
   - `ufit_pro_yearly` — ₹1499/year
   - `ufit_pro_lifetime` — ₹2999 one-time

### Apple In-App Purchase
1. App Store Connect → your app → Features → In-App Purchases
2. Create same 3 product IDs (exact match)
3. Submit for Apple review

---

## Step 6: Run the App

```bash
# Android
flutter run

# iOS (on Mac with Xcode)
flutter run -d ios

# Release APK
flutter build apk --release

# Release iOS
flutter build ios --release
```

## App Architecture Overview

```
lib/
├── main.dart                    App entry, Firebase init
├── app_router.dart              Auth-guarded navigation
├── firebase_options.dart        AUTO-GENERATED by flutterfire configure
├── models/models.dart           All Hive data models
├── providers/
│   ├── app_providers.dart       Habit/Water/Sleep/Weight/Mood/User state
│   └── auth_provider.dart       Firebase auth stream providers
├── services/
│   ├── auth_service.dart        Email, Google, password reset
│   ├── storage_service.dart     Hive local DB init
│   ├── notification_service.dart  Scheduled reminders
│   └── payment_service.dart    Razorpay + IAP
├── screens/
│   ├── auth/
│   │   ├── welcome_screen.dart  Landing page (login/signup choice)
│   │   ├── login_screen.dart    Email + Google login
│   │   ├── signup_screen.dart   Email + Google signup
│   │   └── onboarding_screen.dart  First-time profile setup
│   ├── dashboard/               Main home screen
│   ├── habits/                  Habit tracker
│   ├── water/                   Water intake
│   ├── workout/                 Gym/exercise logging
│   ├── sleep/                   Sleep tracker
│   ├── weight/                  Weight + BMI tracker
│   ├── mood/                    Mood journal
│   ├── premium/                 Paywall
│   ├── settings/
│   │   ├── settings_screen.dart     All app settings
│   │   └── edit_profile_screen.dart  Edit name/weight/height/goal
│   └── profile/                 User profile + achievements
└── theme/app_theme.dart         Full dark/light design system
```

## Auth Flow

```
App Launch
    │
    ▼
Firebase Auth Check
    │
    ├─ Not logged in ──► Welcome Screen ──► Login / Signup ──► Onboarding ──► Dashboard
    │
    └─ Logged in ──► Has local profile? ──► Dashboard
                                        └─ No profile ──► Onboarding
```
